Apache + Perl = mod_perl

Simply install mod_perl and you have a world of features at your fingertips:
  • Configure Apache with Perl. Embed Perl in your httpd.conf file to define everything from virtual hosts to content handlers.
  • Speed up your CGI scripts. The Apache::Registry module can provide 100x speedups for your CGI scripts.
  • Write custom Apache modules. Write simple Perl code to log hits, generate dynamic error pages, track users, and more.
  • Templating systems such as HTML::Mason, EmbPerl, or the Template Toolkit let you quickly develop structured dynamic web sites.
  • Third-party modules give you application functionality such as sessions, passwords, and database integration.
  • Apache 2.0 support includes the ability to write custom protocol handlers in Perl!
